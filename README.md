## Login

| METHOD |        URL       |                 DESCRIPTION                     |
|--------|------------------|-------------------------------------------------|

# Chapters

## List/Details

| METHOD |        URL            |                 DESCRIPTION                     |
|--------|-----------------------|------------------------------------------------ |
| GET    |  /api/allChapters     |  Lista de capítulos                             |
| GET    |  /api/chapter/:id     |  Detalles capítulo                              |

## NewChapter

| METHOD |        URL       |                 DESCRIPTION                     |
|--------|------------------|-------------------------------------------------|
| POST   |  /api/newChapter |  Formulario de nuevo capítulo                   |

## Chapter edit 

| METHOD |        URL             |                 DESCRIPTION                     |
|--------|------------------------|-------------------------------------------------|
| POST   |  /api/chapter/:id/edit |  Formulario de nuevo capítulo                   |
